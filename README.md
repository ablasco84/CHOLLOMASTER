# CholloMaster v2

Agregador automatico de chollos de Amazon España con sistema de alertas por Telegram.

## Stack
- Next.js 14 + React 18
- RSS feeds de Chollometro y Dealabs
- Telegram Bot para alertas
- Vercel (hosting + cron jobs cada 6h)

## Variables de entorno en Vercel
TELEGRAM_BOT_TOKEN=tu_token_del_bot
TELEGRAM_ADMIN_CHAT_ID=tu_chat_id
CRON_SECRET=cualquier_string_secreto
